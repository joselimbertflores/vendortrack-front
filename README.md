# Sistema de Registro y Certificación de Comerciantes - Frontend

Aplicación frontend del sistema de **Registro y Certificación de Comerciantes**.

## Descripción

Este sistema permite registrar comerciantes, gestionar sus puestos de trabajo, categorizarlos por rubro y consultar la validez de certificados emitidos mediante verificación con código QR.

## Funcionalidades principales

- Registro de comerciantes
- Gestión de puestos de trabajo
- Categorización por rubro
- Consulta y validación de certificados
- Verificación mediante código QR

## Requisitos previos

Antes de iniciar, asegúrate de tener instalado:

- Node.js
- npm
- Angular CLI

## Instalación

```bash
npm install
```

## Configuración

Configura las variables y entornos necesarios en los archivos de ambiente del proyecto según corresponda.

## Ejecución del proyecto

### Modo desarrollo

```bash
ng serve
```

### Modo producción
